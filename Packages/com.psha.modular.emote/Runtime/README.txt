Psha VRC Emote Manager

 이 스크립트는 P*Connect 상품을 VRChat 아바타에 손쉽게 도입하기 위한 목적으로 제작된 도우미 툴입니다.
VRCEmote 메뉴(제스처 메뉴의 이모트 슬롯)의 이름, 아이콘, 형식(Button/Toggle)을 빌드 시 자동으로 설정해 주며,
원본 메뉴 에셋을 직접 수정하지 않고 빌드 과정에서만 반영되도록 설계되어 있습니다.

1. 사용 환경 / 필수 의존성

- Unity
  VRChat이 권장하는 Unity 버전 (예: 2022.3.22f1 LTS)

- VRChat SDK
  VRChat SDK3 – Avatars

- 필수 툴
  NDMF (nadena.dev.ndmf)
  이 스크립트의 빌드 패스를 실행하기 위해 필요합니다.

- 선택 툴
  Modular Avatar(MA는 이미 NDMF를 의존하고 있기 때문에, MA가 인스톨 되어있다면 NDMF를 다시 인스톨 할 필요가 없습니다.)
  다른 P*Connect 상품이나 아바타 세팅에서 필요할 수 있습니다.

실제 동작 버전은 상품 페이지에 표기된 지원 버전을 우선으로 해 주세요.

2. 설치 방법

1) NDMF 및 기타 필수 툴 설치
   먼저 NDMF와, 필요하다면 Modular Avatar 등 의존 툴을 프로젝트에 설치합니다.

2) unitypackage 임포트
   Unity 메뉴에서 Assets > Import Package > Custom Package... 를 선택하고,
   P*Connect 상품에 동봉된 unitypackage를 임포트합니다.

3) 프리팹을 아바타에 추가
   패키지 안의 안내에 따라 제공된 프리팹(의상 또는 이모트 세팅용)을
   사용 중인 아바타의 하위 오브젝트로 넣습니다.
   해당 오브젝트에는 Psha VRC Emote Manager 컴포넌트가 붙어 있습니다.

4) 옵션: Setup VRC Emote 실행
   인스펙터에서 Psha VRC Emote Manager의 개발자 옵션에서
   Setup VRC Emote 버튼을 눌러
   타겟 VRC 이모트 메뉴와 아바타 Action/FX 레이어를 자동으로 세팅할 수 있습니다.
   이 단계는 상품에서 제공되는 프리팹에서 설정이 되어있으므로, 기본적으로 실행할 필요가 없습니다.

5) 적용 슬롯 설정
   콘텐츠 내용에 있는 적용 슬롯 NO. (1~8)의 슬라이더를 이동시켜
   VRC Emote 메뉴에 애니메이션을 적용할 슬롯을 지정합니다.
   Emote메뉴는 8개의 슬롯을 제공하고 있습니다.

5) 빌드 및 테스트
   VRChat SDK 컨트롤 패널에서 Build & Test 를 실행하면,
   NDMF 빌드 파이프라인을 통해 VRCEmote 메뉴가 설정대로 적용된 클론 에셋이 생성되고,
   해당 아바타에만 커스터마이즈된 이모트 메뉴가 반영됩니다.
   원본 메뉴 에셋은 수정되지 않습니다.

3. 되돌리는 방법 / 주의사항

1) 커스터마이즈를 끄는 방법
   아바타 하위의 Psha VRC Emote Manager 가 붙어 있는 오브젝트를
   비활성화하거나 삭제한 뒤 다시 빌드하면
   VRCEmote 메뉴는 원래 상태로 빌드됩니다.

2) 여러 개의 매니저를 사용할 때
   같은 아바타 하위에 Psha VRC Emote Manager 가 여러 개 있을 경우,
   하이어라키에서 아래쪽에 있는 컴포넌트부터 적용되고,
   가장 위에 있는 컴포넌트의 설정이 최종적으로 우선합니다.

3) 다른 툴과의 충돌 가능성
   VRCEmote 슬롯을 동시에 수정하는 다른 툴(예: 자체 스크립트, 다른 NDMF 패스 등)을
   함께 사용할 경우 빌드 결과가 예상과 달라질 수 있습니다.
   가능하면 VRCEmote 슬롯 편집 역할은 이 스크립트 한 쪽만 담당하도록 구성해 주세요.

4) 백업 권장
   이 스크립트는 원본 VRCExpressionsMenu 및 Animator Controller를 직접 수정하지 않지만,
   프로젝트 구조나 아바타 설정을 변경하기 전에
   프로젝트와 아바타 프리팹을 미리 백업해 두는 것을 권장합니다.

4. 라이선스 / 재배포 범위

- 본 스크립트는 P*Connect 상품을 자신의 아바타에 도입하기 위한 목적으로만 사용할 수 있습니다.
- 다음과 같은 행위는 허용되지 않습니다.

  1. 스크립트 및 동봉 리소스를 제3자에게 재배포하는 행위
     (무료, 유료를 불문하고 파일 공유 사이트 업로드를 포함)
  2. 본 스크립트를 다른 유료 또는 무료 상품에 포함하여 판매 또는 배포하는 행위
  3. P*Connect 상품과 무관한 프로젝트나 상품에 본 스크립트를 사용하는 행위

각 상품에 적용되는 보다 구체적인 이용 규약(예: VN3 라이선스 등)이 있는 경우
해당 라이선스가 우선 적용됩니다.
자세한 내용은 P*Connect 공통 라이선스 문서를 확인해 주세요.
